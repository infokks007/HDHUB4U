Part 1: Initial Setup (Ek Baar Ka Kaam)

This is the work you only need to do once to get the website online.

Step 1: Database Setup

  1. Go to your hosting control panel (like cPanel) and open phpMyAdmin.
  2. Create a new, empty database (e.g., hdhub_db).
  3. Click on your new database, go to the SQL tab, and run the SQL code provided to create all necessary tables.

Step 2: Upload All Files

Upload all the project files and folders (admin/, config/, uploads/, index.php, etc.) to your website's main directory.

Step 3: Connect to the Database (Most Important)

Open the file: /config/db.php. You must change the details to match your database.

<?php
// --- DATABASE CONNECTION ---
$db_host = 'localhost';
$db_user = 'your_database_username'; // <-- CHANGE THIS
$db_pass = 'your_database_password'; // <-- CHANGE THIS
$db_name = 'hdhub_db';               // <-- CHANGE THIS
?>

Part 2: Admin Panel Ko Kaise Use Karein

Here is how you manage everything on your site.

Step 1: Login

Go to your admin panel URL: www.yoursite.com/admin/

  • Username: admin
  • Password: admin123

Step 2: Add Categories (Menu items)

  1. In the admin panel, click on the Categories tab.
  2. Fill in the form:
    • Button Name: The text that shows on the button (e.g., "Hollywood Movies").
    • Style Class: The button's color (e.g., style-1, style-2).
    • Sort Order: A number to control the order (0 appears first).
  3. Click "Add". This category will now appear on the homepage and in the mobile menu.

Step 3: Add a New Movie

  1. Click on the Add/Edit Movie tab.
  2. Fill in the form:
    • Movie Poster: Upload a poster from your computer.
    • Release Date: Used for the red date tag on the movie post.
    • Quality String: Text below the title (e.g., "480p, 720p, 1080p").
    • Primary Quality Tag: Text for the purple tag (e.g., "720P", "HD-CAM").
    • Categories: Hold Ctrl (or Cmd on Mac) to select multiple categories.
    • Watch & Download Links: Click "Add Link" to create rows for different qualities.
    3. Important: For the "Watch Link", you must use an embeddable URL (like https://www.youtube.com/embed/VIDEO_ID), not a regular watch link.
  3. Click "Save Movie".

Step 4: Search and Edit a Movie

  1. Click on the Movies tab.
  2. Use the Search Bar to find the movie you want to edit.
  3. Click the Edit button. You will be taken to the form with all the movie's information pre-filled.
  4. Make your changes and click "Save Movie".

Part 3: Website Ko Customize Kaise Karein

Logo Kaise Change Karein (How to Change the Logo)

Your website logo is located in the header.php file.

  1. Open the file: header.php.
  2. Find this section of code (around line 21): 
    <div class="header-logo">
  <!-- THIS IS THE LINE YOU NEED TO CHANGE -->
  <a href="/"><img src="https://blogger.googleusercontent.com/...long_link..." alt="Logo"></a>
</div>
  3. To change the logo, replace the link inside the src="..." attribute.

    First, upload your new logo image to your website (e.g., into the uploads/ folder). Then, change the line as shown below.

    Example:

    <!-- FROM THIS: -->
<img src="https://blogger.googleusercontent.com/...long_link..." alt="Logo">

<!-- TO THIS: -->
<img src="/uploads/my-new-logo.png" alt="My Site Name">
  4. Save the header.php file. Your new logo will now appear on every page.
